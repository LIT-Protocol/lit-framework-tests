## Used framework
Svelte

## Framework docs
https://svelte.dev/docs/

## Configuration file
You need to follow these steps for using polyfills with svelte 
1. Configure by replacing the vite.config.ts
2. In src, create a polyfills.ts file
3. In src, create a global.d.ts file

## Getting started
1. `npm i`
2. `npm run dev`
